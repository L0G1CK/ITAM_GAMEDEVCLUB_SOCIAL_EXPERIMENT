# ITAM_GAMEDEVCLUB_SOCIAL_EXPERIMENT
Разработка игры большим сообществом

Смысл данного прооекта - изучить то, как большое сообщество разработчиков без ограничений способно разрабатывать игру.

У всех равные права, есть ограничения на публикуемый контент - сохранение базовых человеческих принципов и соблюдение законодательства РФ, в случае нарушение:
коллаборатор будет заблокирован и не восстановлен.

Правила:
1) Не нарушать базовую инфраструктуру системы(не удалять все коммиты, не заливать сторонний флуд)
2) Использование только Open-Source программ(в нашем случае GODOT, Blender и т.п)
3) Не нарушать моральных принципов

Модератор, наблюдатель за экспериментом - LOGEECK
